# 최근 변경사항 (2026-02-05)

## 🎉 새로운 기능

### 1. 패널 토글
- 중앙 리사이저를 클릭하여 왼쪽 입력 패널을 접거나 펼칠 수 있습니다
- 부드러운 애니메이션 효과
- 상태가 자동으로 저장되어 다음 방문 시에도 유지됩니다

### 2. 변환 이력
- 모든 변환 작업이 자동으로 저장됩니다
- "이력" 버튼을 클릭하여 과거 변환 내역을 확인할 수 있습니다
- 이력을 클릭하면 입력 영역에 즉시 복원됩니다
- 개별 삭제 및 전체 삭제 기능 제공
- IndexedDB를 사용하여 안전하게 저장
